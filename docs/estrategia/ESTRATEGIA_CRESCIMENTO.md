## 📊 Métricas de Acompanhamento (Dashboard Semanal)

### **Métricas de Produto**
- **Clientes Ativos:** Quantos pagam este mês
- **MRR:** Receita recorrente mensal
- **Churn Rate:** % clientes que cancelaram
- **NPS:** Satisfação (pergunta mensal)

### **Métricas de Vendas**
- **Leads/mês:** Quantos prospects novos
- **Conversão:** % leads → clientes
- **CAC:** Custo de aquisição por cliente
- **LTV:** Lifetime value médio

### **Métricas de Operação**
- **Tempo de Onboarding:** Dias até cliente ativo
- **Support Tickets:** Quantos por semana
- **Uptime:** % sistema funcionando
- **Bugs Críticos:** Quantos abertos

**Ferramenta:** Spreadsheet Google (até Mês 12), depois Mixpanel/Amplitude

## ⚠️ Riscos e Planos de Contingência

| Risco | Probabilidade | Impacto | Mitigação |
|-------|---------------|---------|-----------|
| **Churn alto (>20%)** | Média | Alto | Calls de retenção, melhorar onboarding |
| **CAC > LTV** | Média | Alto | Parar ads, focar orgânico/indicações |
| **Concorrente agressivo** | Baixa | Médio | Acelerar MEL (diferencial), nichos específicos |
| **Burnout pessoal** | Média | Alto | Contratar ajuda cedo, férias programadas |
| **Problema técnico grave** | Baixa | Alto | Backup diário, plano de DR (disaster recovery) |
| **Exit prematuro (sem MRR)** | Baixa | Crítico | NÃO SAIR se não tiver 30 clientes + R$ 9k MRR |
