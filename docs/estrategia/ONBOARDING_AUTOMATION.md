**Objetivo:** Cliente ativo em 14 dias, 80% automatizado, sem depender de você em tempo real  
**Criado:** 06/02/2026

---

## 🎯 Filosofia do Onboarding

**Problema Atual:**
- Você tem 16h/semana
- Não pode fazer calls ao vivo com cada cliente
- CLT impede atenção em tempo real

**Solução:**
> **Onboarding assíncrono guiado por MEL + conteúdo gravado**

**Princípios:**
- ✅ Cliente aprende no próprio ritmo
- ✅ MEL guia o processo (mensagens proativas)
- ✅ Vídeos curtos (3-5 min) para cada etapa
- ✅ Gamificação (checklist, badges de conclusão)

---

## 📊 Jornada do Cliente (Day 0 → Day 14)

### **Day 0: Compra + Boas-Vindas**

#### **Evento:** Cliente paga setup (R$ 329) + primeiro mensal

## 🤖 MEL: Scripts de Automação

### **n8n Workflow: Onboarding Day 0-7**

#### **Trigger 1: Cliente Criado**
```
IF cliente.created_at = today
THEN send_whatsapp(boas_vindas_template)
WAIT 6 hours
IF cliente.login_count = 0
THEN send_whatsapp(reminder_primeiro_acesso)
```

#### **Trigger 2: Checklist Item Completado**
```
IF checklist_item.completed = true
THEN send_whatsapp(parabens_template)
CALCULATE next_step
SEND suggestion_next_step
```

#### **Trigger 3: Produto Cadastrado**
```
IF produtos.count = 1
THEN send_whatsapp(incentivo_segundo_produto)
WAIT 1 day
IF produtos.count >= 3
THEN unlock_achievement(badge_catalogo)
```

#### **Trigger 4: Loja Acessada Externamente**
```
IF external_visit.count > 0
THEN send_whatsapp(alerta_visita_loja)
```

#### **Trigger 5: Primeira Venda**
```
IF vendas.count = 1
THEN send_whatsapp(comemoracao_primeira_venda)
SEND email_parabens
```

```

  "to": "{{cliente.whatsapp}}",

```

### **Template: Produto Cadastrado**
```json
{
  "to": "{{cliente.whatsapp}}",
  "message": "Parabéns! Você cadastrou seu {{numero_ordinal}} produto! 🎉\n\n{{#if produtos.count >= 3}}Sua loja já tá show! Quer ativar ela agora?{{else}}Que tal cadastrar mais {{3 - produtos.count}} produtos?{{/if}}"
}
```

### **Template: Primeira Venda**
```json
{
  "to": "{{cliente.whatsapp}}",
  "message": "🎉🎉🎉 PRIMEIRA VENDA ONLINE!\n\nCliente: {{venda.cliente_nome}}\nValor: R$ {{venda.valor}}\n\nParabéns {{cliente.nome}}! Agora é só escalar! 💰"
}
```
