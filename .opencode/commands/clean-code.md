Load clean code principles and coding standards for writing maintainable, professional code.

Execute:
```
skill name="clean-code"
```

The clean-code skill is now loaded and ready to use. It provides:

- **Naming Conventions** - Meaningful names for variables, functions, and classes
- **Function Design** - Single responsibility, small functions, minimal parameters
- **Code Organization** - Structure, formatting, and file organization
- **Comments & Documentation** - When to comment and what to document
- **Refactoring Techniques** - Code smells and how to eliminate them
- **SOLID Principles** - Single responsibility, open/closed, Liskov substitution, interface segregation, dependency inversion

Use this skill when writing new code, refactoring existing code, or conducting code reviews.
