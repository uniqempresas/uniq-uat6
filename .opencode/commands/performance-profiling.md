Load performance profiling techniques and optimization strategies.

Execute:
```
skill name="performance-profiling"
```

The performance-profiling skill is now loaded and ready to use. It provides:

- **Frontend Performance** - Web Vitals (LCP, FID, CLS), bundle optimization, lazy loading
- **Backend Profiling** - Database query optimization, caching strategies, async patterns
- **Bottleneck Identification** - CPU profiling, memory leaks, network analysis
- **Monitoring Tools** - Chrome DevTools, Lighthouse, performance APIs
- **Optimization Techniques** - Code splitting, image optimization, CDN usage
- **Performance Budgets** - Setting and maintaining performance targets

Use this skill when diagnosing performance issues, optimizing applications, or establishing performance monitoring.
