Load API design patterns and best practices for building robust APIs.

Execute:
```
skill name="api-patterns"
```

The api-patterns skill is now loaded and ready to use. It provides:

- **REST API Design** - Resource naming, HTTP methods, status codes, pagination
- **GraphQL Patterns** - Schema design, resolvers, mutations, subscriptions
- **tRPC Integration** - Type-safe APIs with end-to-end type inference
- **Authentication** - JWT, OAuth, API keys, and session management
- **Versioning Strategies** - URL, header, and content negotiation approaches
- **Error Handling** - Standardized error responses and documentation

Use this skill when designing APIs, implementing endpoints, or reviewing API architecture.
