Load systematic debugging methodology for efficient problem-solving.

Execute:
```
skill name="systematic-debugging"
```

The systematic-debugging skill is now loaded and ready to use. It provides:

- **Debugging Methodology** - Reproduce, isolate, diagnose, fix, verify workflow
- **Root Cause Analysis** - 5 Whys, fishbone diagrams, fault tree analysis
- **Debugging Tools** - Debugger usage, logging strategies, tracing
- **Common Patterns** - Heisenbugs, race conditions, memory issues, async bugs
- **Log Analysis** - Structured logging, log levels, distributed tracing
- **Post-Mortems** - Incident analysis and preventive measures

Use this skill when debugging complex issues, investigating production incidents, or establishing debugging workflows.
