Load Docker containerization best practices for production deployments.

Execute:
```

```

The docker-expert skill is now loaded and ready to use. It provides:

- **Dockerfile Best Practices** - Multi-stage builds, layer caching, image optimization
- **Docker Compose** - Service orchestration, networking, volumes, environment variables
- **Production Deployments** - Security hardening, health checks, resource limits
- **Container Patterns** - Sidecar, init containers, ambassador pattern
- **Image Management** - Registry best practices, tagging strategies, scanning
- **Troubleshooting** - Debugging containers, logs, and common issues

Use this skill when containerizing applications, writing Dockerfiles, or managing container deployments.
