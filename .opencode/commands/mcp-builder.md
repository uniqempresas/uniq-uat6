Load MCP builder skill for developing Model Context Protocol servers and tools.

Execute:
```
skill name="mcp-builder"
```

The mcp-builder skill is now loaded and ready to use. It provides:

- **MCP Fundamentals** - Protocol overview, architecture, and core concepts
- **Server Development** - Building MCP servers with tools and resources
- **Tool Definition** - Creating well-designed, documented tools
- **Integration Patterns** - Connecting MCP servers to clients and applications
- **Best Practices** - Error handling, authentication, rate limiting
- **Examples & Templates** - Common patterns and starter implementations

Use this skill when building MCP servers, creating custom tools, or integrating MCP into applications.
