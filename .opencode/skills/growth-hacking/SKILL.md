## 1. Growth Hacking Fundamentals (2026 Edition)

| Campaign-focused | Loop-focused |

| One-time spikes | Compounding returns |

```
Stage 1: LEARN           Stage 2: DEVELOP

        ↓                        ↓
```

## 2. AARRR Framework (Pirate Metrics)

The 5 metrics that matter for sustainable growth.

```
     Acquisition → Activation → Retention → Revenue → Referral
          ↓              ↓             ↓           ↓          ↓
    [Traffic,       [Aha          [DAU/MAU,    [LTV,      [Viral
     signups]        moment,       churn,       ARPU,      coefficient,
                     onboarding]   engagement]  CAC]       NPS]
```

### Acquisition
**Goal**: Get qualified visitors to your product
- **Channels**: SEO, paid ads, content, partnerships, virality
- **Key Metrics**: Traffic, signups, CAC by channel
- **2026 Focus**: AI-optimized paid ads, content-led growth

**Tactics**:
- Programmatic SEO at scale
- Micro-influencer clusters (15-30 small creators > 1 big)
- Community-led acquisition

### Activation
**Goal**: Users experience core value quickly
- **Key Moment**: The "Aha!" moment when value is clear
- **Key Metrics**: Activation rate, time-to-value, onboarding completion
- **2026 Focus**: AI-personalized onboarding

**Tactics**:
- Progressive onboarding (show features as needed)
- Interactive tutorials
- Personalization based on use case

### Retention
**Goal**: Users keep coming back
- **Key Metrics**: DAU/MAU ratio, churn rate, session frequency
- **2026 Focus**: Product-led retention loops

**Tactics**:
- Habit-forming features
- Engagement notifications (not spam)
- Community building
- Regular value delivery

### Revenue
**Goal**: Convert users to paying customers
- **Key Metrics**: LTV, ARPU, conversion rate, payback period
- **2026 Focus**: Usage-based and hybrid pricing

**Tactics**:
- Freemium with clear upgrade triggers
- In-app upsells at value moments
- Transparent pricing

### Referral
**Goal**: Users bring new users
- **Key Metrics**: Viral coefficient (K-factor), NPS, referral rate
- **2026 Focus**: Automated referral loops

**Tactics**:
- In-product referral prompts
- Double-sided rewards (both parties win)
- Viral mechanics (collaboration, sharing)

## 3. Growth Loops

Self-sustaining systems where output is reinvested as input.

### Types of Growth Loops

```
1. VIRAL LOOP (Users bring users)
   
   New User → Uses Product → Shares/Invites → New User
                    ↑_____________________________↓

2. PAID LOOP (Revenue funds acquisition)
   
   Revenue → Ad Spend → New User → Revenue
      ↑________________________________↓

3. UGC LOOP (Content brings users)
   
   User → Creates Content → SEO/Discovery → New User → Creates Content

4. PRODUCT LOOP (Usage drives growth)
   
   User → Uses Feature → Network Effect → More Value → More Usage
```

### Viral Loop Mechanics

**The Viral Coefficient (K)**
- K = Average invitations sent × Conversion rate
- K > 1 = Viral growth
- K < 1 = Need other channels

**Increasing Virality:**
1. **Make it easy to share**: One-click invites
2. **Create value for both sides**: Referrer and referee win
3. **Build virality into product**: Collaboration, visibility
4. **Prompt at the right moment**: After success/achievement

### Loop Optimization Checklist
- [ ] Clear input (what starts the loop)
- [ ] Strong action (what user does)
- [ ] Valuable output (what they get)
- [ ] Reinvestment mechanism (output → new input)
- [ ] Measurable at each step

## 4. Product-Led Growth (PLG)

Growth driven by product usage, not sales teams.

### PLG Principles

1. **End User Focus**: Design for users, not buyers
2. **Low Barrier to Entry**: Free trial or freemium
3. **Value Before Paywall**: Show value before asking for money
4. **Self-Serve**: Users can buy without talking to sales
5. **Viral by Design**: Product naturally spreads

### PLG Funnel

```
Visitor → Signup → Activation → Engagement → Conversion → Expansion
   ↓         ↓          ↓             ↓              ↓           ↓
  SEO    Easy form   Quick       Regular        Upgrade    Upsell/
  Content friction  aha!       usage          triggers   cross-sell
```

### PLG Metrics

| Metric | Definition | Target |
|--------|-----------|--------|
| **Time-to-Value** | Minutes to first aha! | < 5 min |
| **Activation Rate** | % who reach aha! | > 40% |
| **PQLs** | Product-qualified leads | Baseline +10% |
| **Free-to-Paid** | Conversion rate | 5-15% |
| **Net Revenue Retention** | Revenue from existing cohorts | > 100% |

## 5. Experimentation Framework

### The GROWS Method (2026)

**G**ather ideas → **R**ank by ICE → **O**utline experiment → **W**in or learn → **S**hare results

### ICE Scoring

Score each experiment on 1-10:
- **Impact**: How much will this move the needle?
- **Confidence**: How sure are we this will work?
- **Ease**: How easy is it to implement?

**ICE Score = (Impact + Confidence + Ease) / 3**

### Experiment Documentation

```markdown
# Experiment: [Name]

## Hypothesis
If we [change], then [metric] will [increase/decrease] by [amount]
because [reasoning].

## Design
- **Variable**: What we're changing
- **Control**: Current state
- **Sample size**: Number of users
- **Duration**: How long to run

## Success Criteria
- **Primary metric**: Main KPI
- **Target**: Specific goal
- **Minimum detectable effect**: Smallest meaningful change

## Results
- **Outcome**: What happened
- **Learnings**: What we discovered
- **Next steps**: What to do next
```

### Experiment Types

| Type | Use Case | Example |
|------|----------|---------|
| **A/B Test** | Compare 2 versions | Landing page variants |
| **Multivariate** | Test multiple variables | Subject line + CTA color |
| **Funnel Analysis** | Find drop-off points | Onboarding steps |
| **Cohort Analysis** | Track groups over time | Retention by signup month |
| **Channel Test** | Find best acquisition | TikTok vs. LinkedIn |

## 8. Growth Metrics That Matter (2026)

### North Star Metric

**Definition**: The single metric that best captures the core value your product delivers.

**Examples**:
- Airbnb: Nights booked
- Slack: Messages sent
- Spotify: Time spent listening
- Zoom: Minutes hosted

**Choosing Your North Star**:
- Reflects customer value
- Correlates with business success
- Can be influenced by the team

### Cohort Analysis

Track groups of users who signed up in the same period.

```
Cohort    Month 1   Month 2   Month 3   Month 6   Month 12
Jan 2026    100%      45%       35%       28%       22%
Feb 2026    100%      48%       38%       30%        --
Mar 2026    100%      50%       40%        --         --
```

**What to look for**:
- Improving retention over time (product getting better)
- Drop-off points (where users churn)
- Segments that retain better

### Unit Economics

**CAC (Customer Acquisition Cost)**
```
CAC = Total sales + marketing spend / New customers acquired
```

**LTV (Lifetime Value)**
```
LTV = ARPU × Gross margin × Customer lifespan
```

**LTV:CAC Ratio**
- Healthy: 3:1 or higher
- Warning: 1:1 (spending too much)
- Opportunity: 5:1+ (could spend more)

**Payback Period**
- Target: < 12 months
- Excellent: < 6 months
- Dangerous: > 18 months

## 11. Common Growth Mistakes

❌ **Chasing vanity metrics** - Focus on business impact
❌ **Ignoring retention** - Growth without retention is burning money
❌ **Too many channels** - Master 2-3 before expanding
❌ **No experimentation culture** - "We know what works" = stagnation
❌ **Product and marketing silos** - Growth is everyone's job
❌ **Scaling too early** - Don't scale what hasn't been validated
❌ **Copying competitors** - What works for them may not work for you
❌ **Ignoring unit economics** - Growth at any cost is not sustainable
