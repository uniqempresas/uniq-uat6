# Business Model Canvas Frameworks

> Visual tools for designing, testing, and iterating business models on a single page.

---

## 1. Business Model Canvas (Osterwalder)

The classic 9-block framework for describing how a company creates, delivers, and captures value.

```
┌─────────────────────────────────────────────────────────────────┐
│  KEY           │  KEY          │  VALUE          │  CUSTOMER    │
│  PARTNERS      │  ACTIVITIES   │  PROPOSITIONS   │  RELATIONSHIPS│
│                │               │                 │               │
│                │               │                 │  CHANNELS     │
│                │  KEY          │                 │               │
│                │  RESOURCES    │                 │               │
├────────────────┴───────────────┴─────────────────┴───────────────┤
│  COST STRUCTURE                    │  REVENUE STREAMS             │
│                                    │                              │
└────────────────────────────────────┴──────────────────────────────┘
                         CUSTOMER SEGMENTS
```

### The 9 Building Blocks

| Block | Question | Examples |
|-------|----------|----------|
| **Customer Segments** | Who are we creating value for? | Mass market, niche, segmented, diversified |
| **Value Propositions** | What value do we deliver? | Newness, performance, customization, design, price |
| **Channels** | How do we reach customers? | Direct (sales force, web), Indirect (partners, retail) |
| **Customer Relationships** | What type of relationship? | Personal assistance, self-service, automated, community |
| **Revenue Streams** | How do we make money? | Asset sale, subscription, usage fees, licensing, brokerage |
| **Key Resources** | What assets are essential? | Physical, intellectual, human, financial |
| **Key Activities** | What must we do well? | Production, problem solving, platform/network management |
| **Key Partnerships** | Who are our partners? | Strategic alliances, coopetition, joint ventures, buyer-supplier |
| **Cost Structure** | What drives our costs? | Fixed vs. variable, economies of scale/scope |

### Filling the Canvas - Recommended Order

1. **Customer Segments** - Start here
2. **Value Propositions** - For each segment
3. **Channels** - How they buy
4. **Customer Relationships** - How we interact
5. **Revenue Streams** - How we monetize
6. **Key Resources** - What we need
7. **Key Activities** - What we do
8. **Key Partnerships** - Who helps
9. **Cost Structure** - Economics

### Canvas Validation Checklist
- [ ] Each block connects logically to others
- [ ] Value proposition clearly solves customer pain
- [ ] Revenue > Costs (positive unit economics)
- [ ] Competitive differentiation is evident
- [ ] Key resources are actually obtainable

## 2. Lean Canvas (Maurya)

Adaptation of BMC for startups and new products—problem-focused and action-oriented.

```
┌─────────────────┬─────────────────┬─────────────────┬─────────────────┐
│   PROBLEM       │   SOLUTION      │ UNIQUE VALUE    │  UNFAIR         │
│                 │                 │    PROPOSITION  │  ADVANTAGE      │
│                 │                 │                 │                 │
│                 │                 │  HIGH-LEVEL     │                 │
│                 │                 │   CONCEPT       │                 │
├─────────────────┼─────────────────┼─────────────────┴─────────────────┤
│  EXISTING       │   KEY           │             CHANNELS              │
│  ALTERNATIVES   │   METRICS       │                                   │
├─────────────────┴─────────────────┼─────────────────┬─────────────────┤
│  COST STRUCTURE                   │    REVENUE      │  EARLY          │
│                                   │    STRUCTURE    │  ADOPTERS       │
│                                   │                 │  (TARGET        │
│                                   │                 │   CUSTOMERS)    │
└───────────────────────────────────┴─────────────────┴─────────────────┘
```

### Lean Canvas vs. BMC

| Aspect | Lean Canvas | Business Model Canvas |
|--------|-------------|----------------------|
| **Focus** | Problem-solution fit | Business model completeness |
| **Best for** | Startups, new products | Established businesses |
| **Key addition** | Problem, Solution, Unfair Advantage | Partners, Activities, Resources |
| **Risk focus** | High (startup uncertainty) | Moderate |

### Key Elements Explained

**Problem (Top 3)**
- What are the top problems customers face?
- Rank by severity and frequency

**Solution (Top 3)**
- What's the minimum feature set that solves the problem?
- Think MVP, not full product

**Unique Value Proposition (UVP)**
- Single, clear, compelling message
- Answer: "Why is this different and worth buying?"
- Example: "Uber - Your private driver"

**Unfair Advantage**
- Something that cannot be easily copied or bought
- Examples: Insider info, personal authority, community, network effects

**Key Metrics**
- What numbers indicate traction?
- Examples: Acquisition, activation, retention, revenue, referral (AARRR)

**Cost Structure**
- Customer acquisition costs
- Distribution costs
- People, hosting, infrastructure

**Revenue Structure**
- Revenue model(s)
- Lifetime value (LTV)
- Pricing strategy

## 3. Value Proposition Canvas

Deep dive into the fit between customer needs and your offering.

```
┌─────────────────────────────────┬─────────────────────────────────┐
│      CUSTOMER PROFILE           │       VALUE MAP                 │
│                                 │                                 │
│  Jobs-to-be-Done                │   Products & Services           │
│  • Functional                   │   • [List offerings]            │
│  • Social                       │                                 │
│  • Emotional                    │                                 │
│                                 │   Pain Relievers                │
│  Pains                          │   • [How you fix pains]         │
│  • [Bad outcomes]               │                                 │
│  • [Obstacles]                  │                                 │
│  • [Risks]                      │   Gain Creators                 │
│                                 │   • [How you create gains]      │
│  Gains                          │                                 │
│  • [Required outcomes]          │                                 │
│  • [Expected outcomes]          │         FIT                     │
│  • [Desired outcomes]           │         ═══                     │
│  • [Unexpected outcomes]        │                                 │
└─────────────────────────────────┴─────────────────────────────────┘
```

### Customer Profile (Left Side)

**Jobs-to-be-Done**
- Functional: Practical tasks they need to complete
- Social: How they want to be perceived
- Emotional: How they want to feel

**Pains**
- Undesired outcomes, problems, characteristics
- Obstacles preventing job completion
- Risks (financial, social, technical)

**Gains**
- Required: Must-haves
- Expected: Table stakes
- Desired: Would like to have
- Unexpected: Delighters

### Value Map (Right Side)

**Products & Services**
- List all offerings that create value
- Include supporting products/services

**Pain Relievers**
- How specifically do we eliminate pains?
- How do we make things easier?

**Gain Creators**
- How do we create expected outcomes?
- How do we exceed expectations?
- What surprises and delights?

### Testing Value Proposition Fit

**Problem-Solution Fit**
- Does it solve a real, painful problem?
- Do customers acknowledge the pain?

**Product-Market Fit**
- Are customers buying/using it?
- Is usage leading to retention?

**Business Model Fit**
- Can we acquire customers profitably?
- Is LTV > CAC by healthy margin?

## 4. When to Use Each Canvas

| Scenario | Recommended Canvas |
|----------|-------------------|
| New startup idea | Lean Canvas |
| Pivot/existing startup | Lean Canvas |
| Corporate innovation | Lean Canvas or BMC |
| Established business review | Business Model Canvas |
| Product feature prioritization | Value Proposition Canvas |
| Messaging/positioning work | Value Proposition Canvas |
| Investor pitch deck | Lean Canvas → BMC |
| Strategic planning session | Business Model Canvas |

## 5. Canvas Workshop Facilitation

### Preparation
- Print large canvas (A0/A1 size) or use digital tool
- Bring sticky notes in multiple colors
- Invite cross-functional team (max 8 people)
- Allocate 2-4 hours

### Workshop Flow
1. **Individual brainstorming** (15 min) - Everyone fills their own canvas
2. **Share and cluster** (30 min) - Post notes, group similar ideas
3. **Discuss and debate** (60 min) - Challenge assumptions, find gaps
4. **Vote and prioritize** (15 min) - Identify most important elements
5. **Synthesize** (30 min) - Create consolidated canvas
6. **Identify risks** (15 min) - Mark uncertain assumptions
7. **Next steps** (15 min) - Define validation experiments

### Common Mistakes to Avoid
- **Filling boxes in isolation** - Canvas is a system, not checkboxes
- **Being too vague** - "Good service" is not a value proposition
- **Ignoring the numbers** - Revenue and costs matter
- **One canvas for everything** - Different segments may need different canvases
- **Falling in love with first draft** - Iterate based on feedback
